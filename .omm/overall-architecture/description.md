Bipbox is a macOS file-organization app (Swift 6, SPM, macOS 14+) built hexagonally: BipboxCore is a dependency-free domain layer defining ~28 protocol ports (SearchService, VectorIndex, TextEmbedder, KnowledgeStore, FileTextExtracting, ...) plus the domain services that use them. Outer rings implement those ports: BipboxPersistence (SQLite FTS5 search index, SQLite vector index, SQLite knowledge store, JSON stores), BipboxMacOSAdapters (file operations, Vision OCR, folder watcher, security-scoped bookmarks), BipboxMLX (in-process Qwen3-Embedding-0.6B via mlx-swift), and BipboxAI (LLM orchestration, currently no-model defaults). BipboxAppSupport.makeDefault() is the single composition root used by both the GUI app and the headless harness/CLI. BipboxWorkspaceUI renders the library, the Connections semantic graph, and settings; BipboxApp adds the window shell, model-provisioning banner, and lifecycle.
