Sources/BipboxPersistence/ — port implementations over local storage: SQLiteSearchIndex (FTS5 full-text over name/text/tags), SQLiteVectorIndex (vectors per (itemID, modelID) with brute-force cosine nearest), SQLiteKnowledgeStore (items, capture events, relationships, contexts, metadata snapshots), and JSON stores for app settings, sources, rules, and the JSONL activity log. All live under the BIPBOX_DATA_DIR-controlled runtime directory.
