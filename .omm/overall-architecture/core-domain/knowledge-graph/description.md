KnowledgeGraphService assembles the ego/overview graphs the Connections view renders: items, contexts (folders, people, topics), and edges (belongsTo, similar, related). RelatednessService scores lexical similarity from NLTagger tokens; RelatedContextService surfaces context hubs; MissingFileRecoveryService marks items whose file vanished and relinks them when found. IDs are deterministic (KnowledgeIDs) so graphs are stable across rescans.
