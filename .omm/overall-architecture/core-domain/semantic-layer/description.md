The validated topic recipe in pure Swift. TopicDiscovery.discover: mean-center embeddings (anisotropy fix) -> cosine kNN (k=15, parallel via concurrentPerform) -> Louvain (resolution 2.0) -> soft overlapping top-3 membership (threshold 0.30) -> topic edges by centroid cosine (symmetrized top-k). Seeds vs assign split: aggregate units and capped member samples drive discovery, everything else is soft-assigned only. EmbeddingBackfillService re-embeds indexed items missing a vector for the active model (skips dup-tagged items); RetrievalService blends FTS and vector search for semantic answers; TextEmbedder.swift holds the port + EmbedderModelStatus/EmbedderProvisioning and the NLEmbedding fallback embedder used headlessly.
