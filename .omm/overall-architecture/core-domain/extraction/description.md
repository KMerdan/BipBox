DefaultMetadataExtractionService reads plain text and source code (wide extension list) directly as UTF-8 (bounded: 512 KB file cap, 6000-char slice) and delegates rich types to the injected FileTextExtracting port (implemented by MacContentExtractor: PDFKit, NSAttributedString importers, Vision OCR). The result lands in metadata["text.content"], which feeds the FTS index, the embedding text, and IndexedItem.extractedText (used by backfill). Also emits NLTagger tokens/lemmas/names for lexical relatedness.
