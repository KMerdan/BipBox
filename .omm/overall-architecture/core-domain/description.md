Sources/BipboxCore/ — the dependency-free domain (Foundation + CryptoKit + NaturalLanguage only). Defines all protocol ports (ServiceProtocols.swift) and domain models (DomainModels, ServiceModels, SourceModels, MemoryGraphModels, VectorIndexModels), plus the domain services: intake (cold-start scanner, drop intake, organization pipeline, workflow engine, rules), the descent/unit model (FilesystemDescender, UnitTextComposer), extraction (DefaultMetadataExtractionService + FileTextExtracting port), the semantic layer (TextEmbedder/EmbedderProvisioning, TopicDiscovery, EmbeddingBackfillService, RetrievalService, RelatednessService), and the knowledge graph (KnowledgeGraphService, RelatedContextService, MissingFileRecoveryService).
