The GUI entry point (Sources/BipboxApp/BipboxApplication.swift, single file). Builds BipboxAppServices.makeDefault() with an MLXTextEmbedder injected via embedderFactory, hosts RootContainerView (startup-error banner + model-provisioning banner + workspace), and owns ModelProvisioningCoordinator: silent-load if the model is cached, otherwise an explicit Download flow; on becoming ready it triggers embedding backfill + topic recompute. DEBUG hooks: BIPBOX_FAKE_PROVISIONING, BIPBOX_FORCE_STARTUP_ERROR, BIPBOX_DATA_DIR, BIPBOX_CONTROL_API/PORT.
