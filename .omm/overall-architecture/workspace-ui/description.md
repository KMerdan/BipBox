Sources/BipboxWorkspaceUI/ — the SwiftUI workspace: sidebar + library center, the Connections semantic-zoom graph, inspector, sources/rules/activity/settings views. WorkspaceModel (MainActor ObservableObject) is the view-state hub: library search results, selection, group-by lens (Smart/Type/Source/Time), cluster recomputation (Smart = TopicDiscovery off-main), and graph loading through WorkspaceGraphServices. WorkspaceControlServer exposes the DEBUG control API (localhost HTTP) that UI tests drive.
