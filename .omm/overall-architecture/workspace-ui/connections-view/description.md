The semantic-zoom graph (ConnectionsView.swift): Overview (cluster orbs, ids graph.cluster.*) -> cluster zoom (centered node + member file nodes graph.node.item:*) -> file ego (neighbors via KnowledgeGraphService) with breadcrumb back. Edge weights between clusters come from topic-centroid cosine. Accessibility identifiers on leaf elements make the journey XCUITest-drivable (ConnectionsGraphUITests).
