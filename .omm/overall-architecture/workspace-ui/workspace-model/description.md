The MainActor view-state hub (WorkspaceModel.swift, ~700 lines). Owns library results (IndexedItem), selection, the LibraryLens (smart/type/source/time), cached clusters, and topic edges. discoverTopics() builds discovery seeds (unit:project/collection tags + content-bearing loose items + 15-per-collection member sample) vs assign-only vectors from item tags, then runs TopicDiscovery.discover in Task.detached (the O(n^2) kNN must stay off the main actor). clusterLinks() exposes topic-edge weights to the graph renderer.
