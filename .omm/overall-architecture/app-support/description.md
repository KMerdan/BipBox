Sources/BipboxAppSupport/ — the composition root. BipboxAppServices.makeDefault(paths:workflow:embedderFactory:fileManager:) wires every Core port to its implementation (SQLite stores, MacContentExtractor, watcher, embedder — NLEmbedding by default, MLX injected by the app) and exposes the service bundle. Also: BipboxRuntimePaths (data/models dirs, embedder marker), DefaultSourceLifecycleCoordinator (add/pause/remove source -> permission -> initial scan -> watch), WatchFolderAutomationService (arrival handling), PipelineIntakeService, KnowledgeToolRegistrar (AI tool registry).
