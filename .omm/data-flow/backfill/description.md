DefaultEmbeddingBackfillService (actor) — after the model becomes ready (first download or cached load), embeds every indexed item that lacks a vector for embedder.modelID, skipping dup-tagged items. Idempotent; embedText = displayName + extractedText, so unit text and document content flow into the vectors. Triggered by ModelProvisioningCoordinator.onBecameReady, followed by a cluster recompute.
